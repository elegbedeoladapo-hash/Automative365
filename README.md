## What it does

365 Automotive is a premium luxury car club membership platform built for high-net-worth car enthusiasts. Members get access to secure car storage, professional detailing, concierge services, and exclusive events — with tiered membership plans ranging from $2,500 to $10,000/month.

**Live:** [365automotive.vercel.app](https://365automotive.vercel.app)

---

## Features

- 🏎️ **Tiered membership plans** — Bronze, Silver, and Gold tiers ($2,500–$10,000/month)
- 🔐 **Secure authentication** — full login and signup flow
- 🎥 **Video hero background** — cinematic landing experience
- 📅 **Exclusive events** — members-only automotive experiences
- 🔧 **Concierge services** — detailing, storage, and transport
- 📱 **Fully responsive** — optimised for mobile and desktop
- 📧 **Contact system** — enquiry forms for membership applications
